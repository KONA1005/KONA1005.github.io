# 블로그 포스팅 오류_날짜

## 1. 포스팅 오류에 관하여

- 같은날짜에 포스팅을 여러개 했으나 [github.io](http://github.io) 에는 안올라가고, github엔 올라간 상태로 확인

ex) 파일형식 2022-05-03_INSTALL_ORACLE / 2022-05-03_SPARK_INSTALL ...

## 2. 오류 해결

- 해당 포스팅 제목 변경 후 [github.io](http://github.io) 확인 → 정상
